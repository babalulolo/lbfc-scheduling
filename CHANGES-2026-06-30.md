# LBFC Scheduling App — Changes (June 30, 2026)

## 🕐 Pacific time
- [x] App uses Long Beach (Pacific) time everywhere — never UTC or browser time
- [x] Fixed bug where today's shifts dropped off the list after ~5pm
- [x] Clock times, admin time-editing, and the calendar "today" all pinned to Pacific
- [x] Calendar grid shows 12-hour times (1:00 PM) instead of 13:00

## 📊 Google Sheets sync (3 tabs)
- [x] Made the Google Sheet the master file (retired the old .xlsx)
- [x] **Calendar Import** — signups flow in
- [x] **Attendance Tracker** — signups create rows; clock in/out fills Check-in, Check-out, Actual Hours
- [x] **Volunteer Master** — volunteers sync on registration + a "Sync to Google Sheet" button to backfill
- [x] Apps Script backed up in the repo (`google-apps-script/`, secret redacted)

## 👥 User management
- [x] Admins can **block/unblock** volunteers (reversible, keeps history)
- [x] Admins can **remove** volunteers (permanent + clears their sheet rows)
- [x] Guardrails: can't block/remove yourself or the last admin
- [x] Blocked users are bounced at login

## 📅 Shifts
- [x] Edit a recurring shift → apply to **just this one / this + future / all** in the series
- [x] **Add more dates** to any shift (turns a single shift into a series if needed)

## 🚀 Deploy
- [x] All changes pushed to GitHub and live on Railway
